📸 Secure AI-Powered Photo Album

🔧 Architecture Overview

Our photo album system is built with Flask + Gunicorn + SQLite3, deployed via Podman rootless containers. It supports modular expansion and integrates seamlessly with a Telegram bot interface.

📦 Key Components

  • Website Frontend: index.html, upload.html, admin.html, etc.
  • Backend Logic: album_server.py, db_module.py, exif.py, video.py
  • Deployment Scripts: deploy.sh, deploy_all.sh
  • Semantic Search: Independent OpenCLIP-based AI search module
  • Security: TOTP login, session hardening, CSRF protection, and secure cookies

📸 System Feature Overview

  • ✔️ Support for photo uploads (via Telegram Bot / web drag-and-drop)
  • ✔️ Compatible with HEIC / JPEG image formats
  • ✔️ Automatically extracts EXIF data (capture time, GPS, camera details)
  • ✔️ Supports multi-level folder structure (e.g., /cat/2025/Travel)
  • ✔️ Album interface includes category sidebar, search bar, and pagination
  • ✔️ Photo calendar view (calendar.html) shows monthly capture counts
  • ✔️ Photo timeline view (organized by year and month)
  • ✔️ Click a month to preview all images with lightbox zoom and EXIF info
  • ✔️ Modal popup includes ✕ close button in top-right corner for easy exit
  • ✔️ Secure login with password + TOTP two-factor authentication
  • ✔️ Supports Private Mode: restrict photo access to logged-in users only

🛡️ Security Design

  • TOTP + Password login with brute-force prevention
  • Session protection: Secure, HttpOnly, SameSite=Strict
  • Upload filtering: block invalid formats, handle HEIC/HEIF securely
  • Bot Access Control: whitelist, command lockdown, XSS-safe rendering
  • Podman Isolation: rootless containers with secure volume management

💽 Storage Usage Report

The system provides a real-time Storage Report that visually represents how much space each photo category consumes. Inspired by macOS storage bars, it offers a clean and intuitive breakdown of your /photo directory.

  • 📊 Color-coded bar chart showing usage by category
  • 📁 Automatically adapts to any number of dynamic folders
  • 🧠 Ideal for monitoring usage, backup planning, and optimizing space

Whether you're a power user or admin, this tool offers a transparent overview of your storage footprint. Available in the /admin panel.
